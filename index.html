Project Overview

W/AV is a physical music player device that bridges the gap between digital streaming and tactile music control. Built on a Raspberry Pi, it provides an intuitive, hands-on interface for controlling your Spotify library through physical controls.

Key Features

Physical Controls

  • Interlocking Play/Stop buttons
  • Rotary encoder for library navigation
  • 7-band equalizer with slide potentiometers
  • Volume control slider
  • Standby mode toggle switch
  • Waveform visualizer display
  • Headphone jack and RCA outputs

Technical Specs

  • Raspberry Pi-based
  • Spotify integration
  • Custom hardware interface
  • Standby power management
  • Real-time audio visualization
  • IQaudio Pro DAC HAT for high-quality audio output
